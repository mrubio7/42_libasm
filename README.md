## Objetive

Create some functions in assembly language (intel syntax, nasm compiler)

## Use

Uses a `makefile`

`make` compile all -> `libasm.a`.</br>
`make test`compile all and execute the test.
`make clean` clean `*.o`.
`make fclean`clean all.
