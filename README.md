## Objetive

Create some functions in assembly language (intel syntax, nasm compiler)

## Use

Uses a `makefile`

`make` compile all -> `libasm.a`.</br>
`make test` compile all and execute the test.</br>
`make clean` clean `*.o`.</br>
`make fclean` clean all.</br>
